# 项目简介 python417

本项目是一款基于Python开发的图书管理系统，使用Django框架和MySQL数据库。

## 技术栈

- 后端：Python
- 框架：Django
- 数据库：MySQL

## 功能模块

- 图书的增删改查
- 用户注册与登录
- 用户收藏图书功能
- 简单的数据分析

## 系统角色

- 系统管理员：负责图书管理和用户管理
- 普通用户：可注册登录，查看图书，收藏图书
